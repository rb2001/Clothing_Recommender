README 

Project Description: 

Clothing purchase recommender system based on women's e-commerce data. 

The dowloaded file should include the following: 

	CleanedData.csv
	clothing_recommender_system.py
	get-pip.py
	script.py

INSTALL THE FOLLOWING: 

	Python 3.0 or higher
	Visual Studio Code
	Jupyter Notebooks (optional) 
	Streamlit.io


	LIBRARIES: 
	- Numpy
	- Pandas
	- Sklearn
	- matplotlib


COMMANDS TO RUN: 

Terminal: 

streamlit run clothing_recommender_system.py
